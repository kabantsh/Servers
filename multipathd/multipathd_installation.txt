If you want to provision new server for storage then you need multipath service

1- To install multipath service in Redhat based OS:
~# yum install device-mapper-multipath 



2- To install multipath service in Debian based OS:
~# apt install multipath-tools


~# dpkg -i multipath-tools_0.9.4-3+deb12u1_amd64.deb

To download multipath tool for debian
https://debian.pkgs.org/12/debian-main-amd64/multipath-tools_0.9.4-3+deb12u1_amd64.deb.html
