:: Fresh Installation for windows:
:: ===============================

:: To install notepad 
winget install --id Notepad++.Notepad++

:: To install putty
winget install --id XPFNZKSKLBP7RJ

:: To install firefox
winget install --id Mozilla.Firefox

:: To install Git for windows
winget install --id Git.Git

:: To install Git for Docker for Windows
winget install --id Docker.DockerDesktop

:: To install Git for uget
winget install --id uGetdm.uGet


:: To install Google Chrome 
winget install --id Google.Chrome

:: To install libre office
winget install TheDocumentFoundation.LibreOffice

:: To install VLC Media Player
winget install VideoLAN.VLC

:: To install VPN Psiphon
winget install PsiphonInc.Psiphon

:: To install WinRAR
winget install RARLab.WinRAR

:: To install Mozilla Thunderbird
winget install Mozilla.Thunderbird


:: To install Wireshark
winget install WiresharkFoundation.Wireshark


:: To install Npcap as it is required for wireshark
winget install Insecure.Npcap

:: To install Microsoft PowerToys tool for convert image to text in windows
winget install Microsoft.PowerToys


:: To install PowerAutomate tool
winget install Microsoft.PowerAutomateDesktop


:: To install Adobe.Acrobat.Reader
winget install Adobe.Acrobat.Reader.64-bit
