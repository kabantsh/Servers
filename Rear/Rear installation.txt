Rear is natively build in apt repository for Debian based Operating Systems (Ubuntu, Linux Mint, Kali ... etc)

~# apt-get install rear

If you need to install it offline for you can download package from below website and install it
using command :
~# dpkg -i rear_2.6+dfsg-1_amd64.deb
